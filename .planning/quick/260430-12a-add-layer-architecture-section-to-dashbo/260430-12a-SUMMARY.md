## Design choices

- **Markdown table over `st.dataframe`.** Simpler render, avoids the
  Streamlit 1.40.2 `width='stretch'` pitfall, keeps the comparison visually
  uniform with surrounding text.
- **Neutral callout reuses an existing pattern.** Same inline-style override
  (`background:#f3f6f9; border-left-color:#6c757d; color:#495057`) used by the
  residual KPI callout around line 701. No new CSS class.
- **`BAD_SQL` is shown, never executed.** Verified by grep — appears only
  inside `st.code(...)`, never inside `pd.read_sql` / `engine.execute`.
- **Numbering re-pinned monotonic.** Sections now run 1–9 with no duplicate `8.`.
- **Plain ASCII dashes** (`--`) used in code comments and visible markdown to
  stay consistent with project convention (no Unicode in printable code paths).
  HTML callout uses entity escapes (`&minus;`).
